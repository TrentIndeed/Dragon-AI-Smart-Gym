# Dragon Gym

Touch-first React experience for the Dragon Gym smart cable trainer. The layout is optimized for a 1080x1920 vertical screen with large controls for quick adjustments.

## Getting started
1. Install dependencies
   ```bash
   npm install
   ```
2. Run the dev server
   ```bash
   npm run dev
   ```
3. Build for production
   ```bash
   npm run build
   ```

## Features
- Dual motor resistance sliders (0–500 lb) with live velocity caps based on the motor curve.
- Engagement length workflow that temporarily disables resistance while you position the cables.
- Real-time cable length graph with slope and stroke range readouts plus a temporary cable-length slider.
- Touch-friendly workout controls (start workout, start set, reset/retract) and automatic rep counting.
- Force curve and eccentric profile presets with adjustable intensity.
- Exercise selector with preview imagery and a running workout log.
