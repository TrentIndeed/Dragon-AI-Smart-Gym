# Dragon Gym

A blue-themed touchscreen React UI for the Dragon smart gym prototype. The layout is tuned for a 1080x1920 portrait display and surfaces resistance controls, cable telemetry, force curves, workout logging, and exercise previews.

## Scripts

- `npm install` — install dependencies
- `npm run dev` — start the Vite dev server
- `npm run build` — build the production bundle
- `npm run preview` — preview the production build locally
