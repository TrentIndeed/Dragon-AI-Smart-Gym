# Dragon AI Smart Gym

A digital resistance cable machine — a motorized strength-training system with
electronically controlled resistance, driven from a touchscreen app. Set your
target weight in software, get smooth motorized resistance, and track every rep.

## What's here
- **Touchscreen control app** — a Vite + React + Recharts UI (portrait 1080×1920) for setting resistance, viewing live cable telemetry and force curves, previewing exercises, and logging workouts.
- Motorized cable hardware controlled from the app.

> ⚠️ **Heads up:** the working React app currently lives on the `codex/create-dragon-gym-react-app` branch (and related `codex/*` branches), not yet merged into `main`. Check out that branch to run the app.

## Stack
React · Vite · Recharts (touchscreen UI) · motorized cable hardware
